## 背景与适用场景

本文基于实际问题整理：我新装的主机，按电源键之后风扇转一下立刻断电，我一直按着开机键，主板DRAM灯亮。

## 核心结论

- 确认内存插槽是否对应主板推荐的插槽（一般是 2/4 插槽，查主板手册）。
- 你的主板 BIOS 版本可能对新 CPU/高频内存支持不好，尤其是 AMD 新平台（比如 9950X、X870 主板那类）。
- 如果主板出厂 BIOS 过老，可能完全点不亮，需要用 BIOS Flashback 先升级 BIOS。
- 如果装机时 CPU 没放正、针脚/触点有问题，也可能导致内存无法初始化（DRAM 灯亮）。
- 检查 CPU 底部有没有弯针、针脚脏污。

## 详细说明

### 实现思路 1

检查 CPU 底部有没有弯针、针脚脏污

检查 CPU 供电、24Pin、显卡供电是否插牢

### 9950x3d X870 ddr5

检查 24Pin 主供电 + CPU 8Pin/16Pin 是否插牢

好在 X870 主板一般都有 BIOS Flashback 功能，可以不装 CPU/内存，直接用 U 盘更新 BIOS

### 实现思路 3

同时，为 9950X3D 提供专属性能优化，搭配最新 BIOS 和芯片功能，可以带来约 10% 的游戏性能提升

推荐使用 AGESA 1.2.0.3a BIOS，支持 BIOS Flashback / Instant Flash 功能更新
