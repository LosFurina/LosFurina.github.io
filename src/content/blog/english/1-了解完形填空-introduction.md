# 1. 完形填空是什么

- 短文难度通常与教材接近。
- 第一段和首句常用于交代背景或主旨。
- 完形不是孤立单选题，核心是“在语境中选词”。

# 2. 基础策略

- 积累高频词组和固定搭配。
- 先通读再落选项，避免逐空硬填。
- 词汇量的持续积累会带来明显提分。

# 3. 做题流程

## 3.1 先通读，抓主线

- 边读边猜，先确定主题和情感走向。
- 不确定时及时回读，不要死卡单句。

## 3.2 再看选项，逐空定位

- 结合上下文线索做判断。
- 优先用语义、逻辑和搭配排除错误项。

## 3.3 完成后复核

- 检查是否符合全文主旨。
- 检查是否出现语义冲突或语法错误。

# 4. 常见题型

## 4.1 前后呼应

- 通常有同义替换，不会原词复现。

## 4.2 词义辨析

- 选项词性相同、语义相近，重点看语境。

## 4.3 情感态度

- 关注褒义、贬义、中性倾向。

## 4.4 逻辑关系

- 常见因果、并列、转折、否定。

## 4.5 固定搭配

- 常考动宾搭配与介词短语。

## 4.6 主旨判断

- 常见于段尾或文尾句。

## 4.7 基础语法

- 与句意弱相关，重点看结构是否正确。
