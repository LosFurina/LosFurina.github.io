# 句子核心成分

- 主语（名词性质）
- 谓语（动词性质）
- 宾语（名词性质）
- 表语（主语补足语）
- 定语（形容词性质）
- 状语（副词性质）
- 补语

# 常见句型

- 主语 + 谓语
- 主语 + 谓语 + 宾语
- 主语 + 谓语 + 双宾语
- 主语 + 谓语 + 宾语 + 宾补
- 主语 + 系动词 + 表语
