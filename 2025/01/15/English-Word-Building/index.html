派生

Blog: 派生词
Logseq: 派生词

与中文类比

  • 注意：派生并不完全等于构词法

  • 让固定的字母组合表示某种含义，类似于偏旁部首的感觉。例如：

    • 氵：表示和水有关：
      • 江、河、湖、海
    • hydr：表示和水有关：（来自希腊语 water；日耳曼语）
      • ==hydr==ate：水合物
      • ==hydr==ous：含水的

英语词汇通史

  • 日耳曼语词汇多用于日常生活，词汇更加简单，而希腊语、拉丁语和法语的词汇更多出现在正式场合。

    • 希腊语 -> 英语：telephone，grammar
    • 西班牙语 -> 英语：mosquito，tornado
    • 意大利语 -> 英语：lava，volcano
    • 荷兰语 -> 英语：boss，cookie
    • 波斯语 -> 英语：paradise，chess
    • 澳大利亚土著语 -> 英语：kangaroo
    • 阿拉伯语 -> 英语：alcohol，cotton
    • 土耳其语 -> 英语：coffee
    • 日语 -> 英语：kimono（和服），samurai（武士）
    • 印地语 -> 英语：shampoo
    • 因纽特语 -> 英语：kayak（木舟），igloo（圆顶小屋）
    • 中文 -> 英语：茶 -> tea；台风 -> typhoon
    • 英语 -> 中文：poor guy -> 粤语（扑街）；microphone -> 麦克风；disco -> 迪斯科；jazz -> 爵士

  • 这些具有实际意义的外来词汇逐渐成为英语单词的核心，同时能与其他字母组合构成新的单词。

词根词缀

典型举例

  • see ↔ vis：visible, visual, vision
  • hear ↔ aud：audible, audio, audience
  • eat ↔ ed：edible
  • 例子：vis==ible==, aud==ible==, ed==ible==
    • **-ible**：源自古法语，最早来源于拉丁语，表示“能被……的”形容词。

词根（word root）

  • 类似于 vis, aud, ed 的拼写，往往代表或暗示单词的核心意义。
  • 不同的词根可能具有同一个含义：
    • 希腊语：hydr（水）
    • 拉丁语：aqua（水）aquatic（水生的），aquarium（水族馆）
  • 注意：带有某词根拼写的单词并不一定与该词根的含义相关。
    • 例：visa，沙漠

词缀（fix）

  • 词缀能够修饰词根以表达单词意义或属性。

前缀（prefix）

  • 改变单词的词义，但不改变单词的词性。
  • 常见前缀：
    • in-：否定含义
      • invisible, inaudible, inedible
      • 有时也表示“在里面”：
        • inward, inscribe（刻写）
      • 类似否定前缀：
        • im-: impossible
        • un-: uncountable
        • il-: illegal
        • ir-: irregular
        • mis-: mislead
        • dis-: dislike
        • de-: dehydrate
        • non-: nonlinear
    • en-：表示“使得”
      • enlarge, enable

后缀（suffix）

  • 改变词性：同样的词根加不同后缀，核心意思不变，词性改变。
    • visible -> visibility（能见度）

派生组合规则举例

  • 仅有1个词根：
    1. 前缀 + 词根：construct
    2. 词根 + 后缀：structure
    3. 前缀 + 词根 + 后缀：construction
  • 单词包含多个前缀/后缀：
    1. 前缀 + 前缀 + 词根：reconstruct
    2. 词根 + 后缀 + 后缀：structural
    3. 前缀 + 词根 + 后缀 + 后缀：constructional
    4. 前缀 + 前缀 + 词根 + 后缀 + 后缀 + 后缀：reconstructionally
  • 出现两个词根：
    • geostructure
    • geostructural
    • geostructurally
    • pent（五）+ a + gon（角）：pentagon
    • maintain -> maintenance
    • describe -> description

派生规则的特殊情况

  • 如果词根是一个已有的单词（通常来源于法语），一般可加 -able：
    • count -> countable
    • work -> workable
    • accept -> acceptable
    • remark -> remarkable
  • 如果词根来源于希腊/拉丁，通常加 -ible：
    • credible, terrible, possible
  • 特例：
    • 拉丁词根 eff 在英语中只出现于 ineffable（不可描述的），而 effable 虽未被词典收录，但已被广泛使用；effible 则不存在。

合成

  • 简单单词合并构词，词性多样：
    • waterfall（n.），waterproof（adj.），underwater（adj.）
    • outbreak（n.），one-way（adj.），high-class（adj.）
    • beforehand（adv. 预先，事先）
    • smoke + fog = smog
    • smoke + haze = smaze
    • snow + ice = snice
    • web + site = website
    • website + seminar = webinar
  • 派生构词与合成构词往往有同义或近义词：
    • leg（读）-> legible（字迹清楚）
    • read -> readable（语句通顺）

混合

  • 更随意的合成构词：
    • brunch: breakfast + lunch

截短

  • 为了简化拼读，常掐头去尾：
    • telephone -> phone
    • airplane -> plane
    • mathematics -> math
    • examination -> exam
    • application -> app
    • influenza -> flu
    • refrigerator -> fridge

缩写

  • 用单词首字母组合成一个词：
    • VIP: Very Important Person
    • IELTS: International English Language Testing System
    • TOEFL: Test of English as a Foreign Language
    • MVP: Most Valuable Player
    • ETC: Electronic Toll Collection

转化

  • 一个单词具有多个词性：
    • name（v. n.）
    • fine（adj. v. n.）

Why do you think the English Tenses is so confused?
时间&状态未分清学校教授时态时，往往是将时间和状态混合在一起教，导致学生对时态仅有一个模糊的整体概念。或者即使区分，也并不特别指出。这就导致很多人脑种的时态概念是这样的： 这解释了为什么在学校学到的英语会强行加入很多语法规则，什么时候必须用一般将来，什么时候用过去将来完成时等等。规则复杂没有统一的规律绝对是大多数人在学习英语时被劝退的头号通缉犯。但是如果你问Native English Speakers，他们会说根本不知道什么是时态，就是怎么自然怎么说。 因此，我们需要知道时间和状态其实是两个概念，而完整的时态是由时间和状态排列组合而成的。 动词变位（Verb Conjugation）单词拼写往往会改变。 在中文里表达时间概念时，只需要加上对应的时间相关的描述词即可，不需要对字本省做出任何改动，比如： 我已经吃过饭了 我正在跑步但是在英文中，我们不仅需要加上对应的时间相关的描述词，还需要对动词本身的拼写做出改变， 比如： I ate an apple. （时态变位） She eats an apple. （人称变位） I have eaten an apple....
